# NPA Biblically Responsible Investing

Landing page for Northeast Planning Associates' Biblically Responsible Investing (BRI) offering.

## Contents

- `index.html` — self-contained single-page landing site (inline CSS, embedded images). No build step required.

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Hosting

Deployed on Netlify as a static site. The repository root is published as-is; `index.html` is the entry point.
